# tcpBBC

chat online system base on tcp

### Quick Start 

    Example on windows:
    server:
        cd /server
        go build
        ./server.exe  or ./server.exe -port 9091

    client:
        cd /client 
        go build 
        ./client -ip 127.0.0.1:9091
        Then follow the prompts.
